# Tool Specification Template

## Purpose
Standardized template for documenting tools used by Sol AI assistant.

## Template

```markdown
## Tool Name
**Purpose:** One-line description of what this tool does

**When to use:**
- Use case 1
- Use case 2
- Use case 3

**Inputs:**
- `param1`: description (type, required/optional)
- `param2`: description (type, required/optional)

**Outputs:**
- Description of what this tool returns

**Examples:**
```
Example command or usage
```

**Pitfalls:**
- Common mistake 1
- Common mistake 2

**Related tools:**
- Tool A (when to use instead)
- Tool B (complementary)
```

## Usage
For each tool in `tools/`, create a markdown file using this template.
